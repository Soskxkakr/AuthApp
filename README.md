# AuthApp

A React Native authentication application with user registration, login, and session management.

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/Soskxkakr/AuthApp.git
```

2. Navigate to the project directory:
```bash
cd AuthApp
```

3. Install dependencies:
```bash
npm install
```

4. Run the app:
- For Android:
```bash
npm run android
```
- For iOS:
```bash
npm run ios
```

## Requirements
- Node.js version 18.x or higher
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development)

## Features
- User registration with email and password
- User login
- Persistent session management
- Material Design UI using React Native Paper
- Form validation and error handling
- Password visibility toggle
- SafeAreaView for proper layout

## Tech Stack
- React Native
- React Native Paper
- AsyncStorage
- React Navigation
